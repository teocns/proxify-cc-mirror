## Slash Commands

| Command | What it does |
|---------|-------------|
| `/compact` | Summarize conversation, free context |
| `/plan` | Enter plan mode (no edits until approved) |
| `/copy` | Copy last response to clipboard |
| `/fork` | Fork the current conversation |
| `/chrome` | Toggle native browser integration |
| `/usage` | Show rate limits and usage |
| `/stats` | GitHub-style activity graph |
| `/clear` | Clear conversation, start fresh |
| `/mcp` | Manage MCP server connections |
| `/config` | Open configuration menu |
| `/statusline` | Configure status line display |

## Keyboard Shortcuts

| Shortcut | What it does |
|----------|-------------|
| Shift+Tab | Toggle plan mode |
| Cmd+A / Ctrl+A | Select all (useful for copy-pasting content to Claude) |
| fn fn (macOS) | Trigger dictation |
| Esc | Cancel current generation |

## CLI Flags

| Flag | What it does |
|------|-------------|
| `-c` / `--continue` | Continue last conversation |
| `-r` / `--resume` | Pick a recent conversation to resume |
| `--fork-session` | Fork when resuming/continuing |
| `--chrome` | Start with Chrome integration enabled |
| `-p "prompt"` | Non-interactive single prompt |
| `--dangerously-skip-permissions` | Auto-approve all tool use (use in containers only) |

## Shell Aliases Worth Setting Up

```bash
alias c='claude'
alias ch='claude --chrome'
```
